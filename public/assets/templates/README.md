# Quote and NRL specification placeholders

- **quote-template.pdf** – Minimal valid PDF placeholder for the formal quote (e.g. 41260018 NRL). Replace with the real quote PDF for production.
- **nrl-template.xlsx** – Placeholder for the NRL specification sheet. For production, replace with the actual Excel spec sheet (e.g. 41260018 NRL.xlsx) or generate it from your data.

The add-in can attach these files when sending the final quote to the customer. Ensure the server serves this directory if the add-in fetches templates by URL.
